# Historical Records

Documents in this directory are **frozen**: they record the project's planning and review
history exactly as approved, and are never edited. Their living content has been extracted
into the canonical documents (`PROJECT_CONSTITUTION.md`, `ARCHITECTURE.md`, `ROADMAP.md`,
`STYLEGUIDE.md`, ADRs, `docs/*.md`). When history and a canonical document disagree, the
canonical document is current truth.

| File | What it is |
|---|---|
| `2026-07-engineering-plan-v1.1.md` | The approved engineering plan (v1.1, post-review). Source of most canonical docs. |
| `2026-07-architecture-review.md` | Principal Engineer architecture review of plan v1.0. |
| `2026-07-architecture-review-reconciliation.md` | Disposition of every review finding; rationale for accepted/rejected items. |
| `2026-07-m0-bootstrap-plan.md` | Repository bootstrap + M0 commit-by-commit plan (this milestone's execution contract). |
